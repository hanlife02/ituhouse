# 小兔书 ituhouse

前后端分别位于 `backend/`（FastAPI）与 `frontend/`（Next.js）。

## Backend（FastAPI）

```bash
cd backend
pip install -r requirements.txt
cp .env.example .env
cp email_senders.json.example email_senders.json
python main.py
```

## Frontend（Next.js）

```bash
cd frontend
pnpm install
pnpm dev
```
